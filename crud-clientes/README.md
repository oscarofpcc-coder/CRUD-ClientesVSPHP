# CRUD Clientes – PHP con VS 2026

Proyecto CRUD de clientes usando:

- Frontend: PHP + HTML + CSS + VS 2026
- Backend: API REST en C# ASP.NET Core
- Base de datos: MySQL (MAMP)
- Arquitectura: POO + API 

## Estructura
- frontend-php/: interfaz web con modales 
- backend-api/: API REST en VS

## Requisitos
- PHP 8+
- MAMP (MySQL)
- .NET 10 o superior
- Git

## Ejecución
1. Ejecutar la API en Visual Studio
2. Ajustar API_BASE en http://localhost:5286
3. Abrir http://localhost/crud-clientes/index.php


## API REST ASP.NET Core.
Endpoints:
- GET /api/clientes
- POST /api/clientes
- PUT /api/clientes/{id}
- DELETE /api/clientes/{id}
